# Public-transport-tracking
This real time public transport tracking system app mainly focuses on tracking of public buses that arrive to the customer's location accordingly. It also displays the price of the ticket beforehand so that people don't have to be in a confused state about the fare.
The additional feature includes booking of a cab or autorickshaw whenever required.
